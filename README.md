# Figma Backup

Скачивает `.fig` из Figma и загружает в папку Google Drive. Управление — через веб-интерфейс.

**Требования:** macOS, Node.js 18+, Google Chrome.

## Скачивание

Скачайте ZIP с GitHub и распакуйте.

При первом запуске через `start.command` зависимости установятся автоматически.

## Запуск

**macOS (двойной клик):** `start.command`
Откроется http://localhost:3847

## Использование

1. Нажмите **Войти в Figma** и **Войти в Google Drive** в одном окне Chrome.
2. Добавьте ссылки на проекты Figma и папки Drive.
3. Отметьте нужные строки и нажмите **Сделать бэкап**.

Не закрывайте Chrome до завершения бэкапа. Скачанные файлы сохраняются в `backup/` (с перезаписью), список ссылок — в `links.json`.
